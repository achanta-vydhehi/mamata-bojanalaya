# 🍽️ Mamta Bhojnalaya – Responsive Restaurant Website

## 📌 Project Overview

Mamta Bhojnalaya is a responsive restaurant website developed as part of a Web Development Internship. The project showcases a modern and user-friendly restaurant website with multiple sections, including Home, About, Menu, Gallery, Customer Reviews, and Contact.

The website is designed using HTML5, CSS3, and JavaScript while maintaining clean code and a responsive layout for different screen sizes.
